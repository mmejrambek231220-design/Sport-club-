## 1fit.app — Зал беті (club-detail) үлгісі

> Зерттелген бет: https://1fit.app/ru/astana/studios/4435 (Yogalife)  
> Playwright арқылы скриншот алынып талданды — 2026-04-09

### Беттің жалпы орналасуы (3 баған)

```
┌──────────────┬─────────────────────┬──────────────────────────┐
│  СОЛ ЖАҚ     │   ОРТА ПАНЕЛЬ       │      ОҢ ЖАҚ              │
│  Залдар      │   (зал детайлы)     │      2GIS Карта          │
│  тізімі      │   ~300px кеңдік     │      (толық биіктік)     │
│  sidebar     │   scroll жасалады   │                          │
└──────────────┴─────────────────────┴──────────────────────────┘
```

**Қай жерден ашылады:**  
- Сол жақ тізімдегі зал карточкасына басқанда → ОРТА ПАНЕЛЬ ашылады  
- URL өзгереді: `.../studios/{id}` — тікелей сілтемемен де ашуға болады  
- Карта маркеріне басқанда тек поп-ап ашылады (панель емес)

---

### ОРТА ПАНЕЛЬ мазмұны (жоғарыдан төменге)

#### 1. Жабу батырмасы
- **[✕]** — оң жоғарғы бұрышта, панельді жабады, тізімге қайтады

#### 2. Фото галерея (carousel)
- Бірнеше фото горизонталь слайдер ретінде
- **[←] [→]** — солға/оңға навигация стрелкалары
- Фотолар үлкен, панельдің толық кеңдігіне созылған

#### 3. Зал атауы + рейтинг
- **Атауы:** үлкен, қалың (мысал: "Yogalife")
- **Рейтинг:** жасыл/сары бейдж — сан (мысал: "10", "9.9")
- **Категория тегі:** "Студия групповых занятий"
- **Мекенжай:** 📍 иконкасымен (мысал: "улица Санжара Асфендиярова, 4")

#### 4. "О зале" бөлімі
- Зал туралы қысқа сипаттама мәтіні (2-3 жол)
- **"Посмотреть ещё"** — мәтінді толық ашатын сілтеме

#### 5. "Виды занятий" бөлімі
- Тег/чип ретінде жаттығу түрлері:
  ```
  [Йога]  [Другие виды йоги]  [Стретчинг и пилатес]  [Танцы]
  ```
- Чиптер басылмайды, тек ақпараттық

#### 6. "Расположение" бөлімі
- Мини Google карта (embed)
- Залдың орналасуы маркермен белгіленген
- Кіші масштаб (~200м радиус)

#### 7. "Удобства" бөлімі (Amenities)
Иконка + мәтін форматында, 2 бағанды тор:

| Белгі | Қолжетімді | Белгі | Қолжетімді |
|-------|-----------|-------|-----------|
| 💧 Питьевая вода | ✅ | 🚿 Душ | ✅ |
| 🚗 Парковка | ✅ | 📶 Бесплатный Wi-Fi | ✅ |
| 🎵 Музыка в зале | ✅ | ❄️ Кондиционеры | ✅ |
| 🧖 Сауна | ❌ (сұр) | 🚇 Метро рядом | ❌ (сұр) |
| 🕌 Намазхана | ❌ (сұр) | 💇 Фен | ❌ (сұр) |
| 🍵 Бесплатный чай | ❌ (сұр) | 🛁 Бесплатные полотенца | ❌ (сұр) |
